Timestamp Microservice (FCC API Project)
=========================

[It's live!](https://bow-passenger.gomix.me)

### Example usage

`https://bow-passenger.gomix.me/December%2015,%202015`

Or

`https://bow-passenger.gomix.me/1450137600`

#### Example output

```json
{
  "unix": 1450137600,
  "natural": "December 15, 2015"
}
```
